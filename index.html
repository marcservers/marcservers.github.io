How to join the server on Java and Bedrock edition.

Java Edition

marcsmp.abrdns.com
  • Open Minecraft Java Edition.
  • Click Multiplayer.
  • Click Add Server.
  • Enter the IP address above and set the name to MarcSMP.
  • Click Done and join.

Bedrock Edition

bedrock.marcsmp.abrdns.com
  • Open Minecraft.
  • Go to the Creator Experience.
  • Click Add Server.
  • Enter the IP address above and set the name to MarcSMP.
  • Change the port to 61060 and click Play.
